The Humanity node

The ultimate object: a small computer in weatherproof metal, engraved with the same link and QR as a sticker—but also able to tap, relay, and uplink when the grid is thin.

  • Raspberry Pi 5 (edge compute) Runs resolver client, mesh bridge, and local cache of signed status—not a second trust root
  • NFC front-end NTAG or equivalent — tap opens the same /c/… URL
  • QR + engraved URL Laser-etched link on metal housing; camera path unchanged for strangers
  • Solar + battery Outdoor install without running conduit; sleep/wake duty cycle TBD
  • Metal enclosure Machined or cast shell — the node is meant to be kept, mounted, or bolted in place

Connectivity stack (later)

  • Cellular modem LTE for resolver sync where Wi‑Fi never reaches
  • Satellite backhaul (research) Starlink or similar when the node must stay honest off-grid—we have not picked hardware or cost model
  • Bluetooth mesh Node as relay between tap and phones in a crowd

Open questions: firmware update path, who operates the node, key custody on device, and whether the node ever signs on behalf of the owner or only caches public resolver state.
