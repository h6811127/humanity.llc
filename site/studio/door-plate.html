Pilot object

Why we started with a studio door plate

A workshop door is the smallest honest test of “live state on a physical object.” Visitors only need one answer: are you open right now? A paper sign lies by afternoon. A QR can tell the truth at scan time.

  • Object name — which studio or room (fixed on the card).
  • Status line — hours, open, closed, or done for the night (editable).
  • Same sticker — update the line from /created/ without reprinting.

We use this plate on our own door first. Extensions: retail hours, festival booths, critique nights, shared-building wayfinding, and “closed for install week” — same template, different copy.
