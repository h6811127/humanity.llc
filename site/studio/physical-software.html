Framing

Printing is deployment

A QR on a door is not a link on a flyer. It is a deployed endpoint: the resolver returns current signed state when someone scans. Revoke changes the answer; the ink stays.

The studio status plate is the teaching example - small surface area, daily use, obvious when the line is wrong. From there the same object model stretches to wearables, benches, and crisis cards (research tracks, not all shipped).
