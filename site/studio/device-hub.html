On this device - keys without accounts

humanity.llc deliberately has no login. Signing keys live in the browser. That split is confusing until the UI names it:

  • This tab - sessionStorage right after create.
  • This device - localStorage after you tap Save on this device.

The homepage On this device hub is the control center: saved cards with Use keys / Open scan, pinned public scans, shortcuts, backup import, and search that filters only local rows - not the public resolver.

Search moved inside the hub (not a floating button) so it stays next to what it filters. Shortcuts and backup stay visible even when a filter matches nothing - so clearing search on iPad cannot hide your way back to import or revoke flows.
