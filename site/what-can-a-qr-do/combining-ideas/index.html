Authority stays clear

Games + maintenance

  • Game layer rewards discovery and reports
  • Maintainer layer signs in/out-of-service
  • Map unlock reads signed maintenance state

Crisis + freshness

  • Aid layer signs access and location info
  • Kitchen layer signs made, expires, allergens
  • Recall flips safety without user tracking

Markets + surplus

  • Co-op layer signs stall membership state
  • Vendor layer signs menu and sold-out state
  • Prepared-food tags sign safe-to-take window

Best combos keep authority clear: the game can reward discovery, but only the maintainer signs safety; the crisis card can show food, but the kitchen signs freshness.
