Core combo patterns

Games + maintenance

  • Players find AED, filter, and trail-marker nodes
  • Maintainer-signed status stays separate from game points
  • Fix verified → local map unlocks the next route

Crisis cards + freshness

  • Wallet aid card lists nearby pantry and fridge options
  • Each food batch keeps made, expires, allergen, recall state
  • Resource page updates without tracking who scanned

Markets + menus + surplus

  • Co-op pass links to today’s stalls and live menus
  • Prepared-food tags mark safe-to-take surplus
  • Vendor updates one state object instead of reprinting

Best combos keep authority clear: the game can reward discovery, but only the maintainer signs safety; the crisis card can show food, but the kitchen signs freshness.
