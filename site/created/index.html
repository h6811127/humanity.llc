Your card is controlled by a private signing key generated in this browser. It never uploads to humanity.llc. The resolver only sees signed actions (like revoke).

  • This browser tab keeps the key until you close it. Add to Home Screen does not save the key.
  • Recovery key (shown once at create) or encrypted backup (below) lets you revoke from another device later.
  • iPhone (Passwords) or Android (Google Password Manager): save the backup passphrase as a secure note or login entry for humanity.llc backup. When importing, tap the passphrase field and pick that entry—autofill often fills the wrong field or leaves it empty.
  • Do not store the .hcbackup.json file in a shared folder without understanding anyone with the file can try to guess your passphrase.
